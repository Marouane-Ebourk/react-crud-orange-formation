# React + Json-server + Tailwind

To setup the project:
  ```bash
    git clone https://github.com/Marouane-Ebourk/react-crud-orange-formation.git
  ```
  ```bash
    npm install
    ``` 
To start the server:
  ```bash
    json-server --watch db.json
  ```
To run the project:
  ```bash
    npm run dev
  ```
